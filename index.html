Data Hygiene Rules

  • Runner/source model means who produced the batch.
  • Payload label means what the model wrote under “MODEL TESTED.”
  • Do not call a Claude-run simulation “Gemini evidence” just because the payload says Gemini.
  • Use Outcome preference, not score alone. Sometimes both score high but one is cleaner.
  • Use card weight to decide whether the fix needs Full, Mini, or Direct.

Import / Export

Import accepts the V3 JSON format: either an array of records or an object with a records array. The parser above can also preview rough records from pasted batch output.
