# 02 — GitHub Setup

## 1. Create the repository

Go to GitHub and create a new repository.

Recommended repo name:

```text
harness-card-v3
```

Recommended description:

```text
Local-first AI chat circuit breaker for recovering drifting, looping, overexplaining, or ungrounded AI conversations.
```

Initial visibility:

```text
Private
```

Do not initialize with README, license, or `.gitignore` because this repo already includes those files.

## 2. Upload files

Unzip the latest repo ZIP.

Upload the contents of the repo folder, not the ZIP file itself.

Upload these root files and folders:

```text
index.html
README.md
DISCLAIMER.md
PRIVACY.md
SECURITY.md
LICENSE
package.json
.gitignore
REPO_MANIFEST.txt
data/
docs/
.github/
roadmap/
```

## 3. Commit message

Use:

```text
Initial Harness Card V3 prototype
```

## 4. Verify GitHub display

After upload, confirm GitHub shows:

- README rendered on the repo front page
- `DISCLAIMER.md`
- `PRIVACY.md`
- `SECURITY.md`
- `docs/`
- `roadmap/`
- `GITHUB_ISSUE_TEMPLATE_test-report.yml (visible reference file; optional later GitHub path is .github/ISSUE_TEMPLATE/test-report.yml)`

## 5. Do not make public yet

Keep it private until GitHub Pages or local testing confirms the demo works.
