# Roadmap

This folder is the practical release roadmap for Harness Card V3.

The goal is simple:

> Get the local-first prototype onto GitHub, make the demo link work, collect sanitized tester feedback, and improve the product without collecting private user data.

Current status:

- Project/build name: Harness Card V3
- Public positioning: AI chat circuit breaker
- Naming: working name, not final
- Product status: early local-first prototype
- Evidence policy: collect repair outcomes, not personal conversations
- Next product priority: clipboard parser / auto-log improvement

Follow these files in order:

1. `01_PRE_PUBLIC_CHECKLIST.md`
2. `02_GITHUB_SETUP.md`
3. `03_GITHUB_PAGES_DEMO.md`
4. `04_FEEDBACK_COLLECTION.md`
5. `05_FIRST_PUBLIC_POSTS.md`
6. `06_NEXT_BUILD_BACKLOG.md`
7. `07_GO_NO_GO_CHECKLIST.md`

Do not skip the privacy and disclaimer checks before making the repository public.
