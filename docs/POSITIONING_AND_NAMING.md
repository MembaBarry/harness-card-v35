## Core Product Definition

Harness Card V3 is a local-first AI conversation repair tool.

When an AI chat drifts, forgets constraints, restarts, overexplains, loops, or makes unsupported claims, the tool generates the right-size correction prompt: Full Harness, Mini Harness, or Direct Correction.

It works across AI systems because it is plain structured language.

No account. No API key. No backend. No analytics.

## Core Thesis

Prompt engineering helps users ask better at the beginning.

Harness Card V3 helps users recover when the conversation starts losing the plot in the middle.

This is context repair / conversation repair, not just prompt writing.

## Strongest Honest Claim

Early cross-model batch testing suggests Full Harness Cards outperform weak baseline corrections on layered context failures such as lost constraints, drift, restart loops, fake certainty, and trust/handoff problems.

For simple output-format failures such as “code only” or “no explanation,” Mini Harness or Direct Correction may be better unless the model keeps failing after correction.

Do not claim proof, universality, novelty, or that no competitor exists.

## Public Pitch Under Review

Harness Card V3 is a local-first AI chat circuit breaker for recovering drifting, looping, overexplaining, or ungrounded conversations.

## Technical Abstract Under Review

Harness Card V3 is a local-first context-repair utility for human-AI interaction. It generates structured diagnostic prompts that help recover multi-turn AI conversations from drift, lost constraints, restart loops, unsupported certainty, overexplaining, and handoff failure. Early batch testing across multiple AI systems suggests full diagnostic cards improve layered context recovery, while lighter mini-cards or direct corrections are better suited for simple output-format violations.

## What Not To Build Yet

- accounts/login
- Stripe/payments
- cloud sync
- custom LLM runner
- automated API logging system

The current strength is a free, local-first, copy-paste utility with evidence tracking.

## Best Next Feature

The clearest next feature is the V4 friction fix:

> Paste raw thread or batch result → scrub UI noise → estimate context weight → recommend Full/Mini/Direct → preview/import records.

This reduces manual-entry pain while preserving human review before evidence is logged.
