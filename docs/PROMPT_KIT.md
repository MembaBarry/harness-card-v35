# Prompt Kit

## Full Harness Spine

```text
STOP.

Before continuing, output these labels exactly:

POINT A:
POINT B:
DRIFT:
LOST CONSTRAINTS:
UNDERLYING SIGNAL:
TRUST ISSUE:
HANDOFF RISK:
NEXT STRAIGHT STEP:

Then continue with the corrected answer.
```

## Mini Harness — Output Discipline

```text
MINI HARNESS — OUTPUT DISCIPLINE

Stop.

You violated the requested output format.

Original request:
[what I asked for]

Violation:
[what you added, changed, or ignored]

Now answer again using only the requested format.

No explanation.
No intro.
No outro.
No diagnostic labels.
No extra text.
```

## Direct Correction

```text
You missed the point.

Original request:
[what I asked for]

What went wrong:
[what you did wrong]

Correct it now by doing only this:
[short exact correction]
```
