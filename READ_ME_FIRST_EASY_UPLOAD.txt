HARNESS CARD V3 — EASY GITHUB UPLOAD PACKAGE

This folder intentionally contains NO hidden dot-files or dot-folders.

Upload everything inside this folder to GitHub using Add file → Upload files.

This package still includes:
- the working app: index.html
- README.md
- DISCLAIMER.md
- PRIVACY.md
- SECURITY.md
- LICENSE
- data/
- docs/
- roadmap/

Optional later:
- GITHUB_ISSUE_TEMPLATE_test-report.yml contains the GitHub issue template text.
  If you want GitHub's automatic issue form later, create this path manually in GitHub:
  GITHUB_ISSUE_TEMPLATE_test-report.yml (visible reference file; optional later GitHub path is .github/ISSUE_TEMPLATE/test-report.yml)
  and paste the contents of GITHUB_ISSUE_TEMPLATE_test-report.yml.

- GITIGNORE_REFERENCE.txt contains the .gitignore rules.
  If you want a real .gitignore later, create a file named .gitignore in GitHub
  and paste the contents of GITIGNORE_REFERENCE.txt.

You do NOT need those optional hidden files to run the demo or publish the repo.
